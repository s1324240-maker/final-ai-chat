# リアルタイム翻訳アプリ

マイクに向かって話すと音声を認識し、指定した言語へすぐに翻訳して、翻訳結果を文字と音声の両方で確認できるWebアプリです。外国人との会話、海外旅行、留学などの場面で、スムーズにコミュニケーションを取ることを目的としています。

単語をそのまま置き換えるだけではなく、会話で使いやすい自然で分かりやすい表現に翻訳できる体験を目指しています。文字入力からの翻訳と送信にも対応しており、同じURLを開いた相手とリアルタイムに会話できます。

## 主な機能

- マイク音声のリアルタイム文字起こし
- 文字入力からの翻訳
- Socket.IO による複数人のリアルタイム会話
- 受信メッセージを自分が選んだ言語へ自動翻訳
- 日本語、英語、韓国語、中国語、スペイン語、フランス語など32言語に対応
- 翻訳元と翻訳先の入れ替え
- 入力テキストのお手本読み上げ
- 翻訳結果の読み上げ
- 発音した言葉のネイティブ度スコア表示
- 翻訳結果のコピー
- 翻訳履歴の表示
- お気に入り登録
- よく使うフレーズ集
- 会話スターター

## 利用シーン

- 外国人との日常会話
- 海外旅行での質問や案内
- 留学先での会話サポート
- 多言語メンバーとの簡単な打ち合わせ

## 作品制作でアップデートした内容

- ITパスポート出題アプリのサンプルから、会話サポート型のAI活用アプリへ変更
- `promt.md` にアプリの目的、AI活用ポイント、デプロイ手順を記録
- Render.comで起動しやすいように `npm start` で `server.js` が起動する構成に整理
- 旅行、学校、買い物などの場面で使えるフレーズ集を追加

## セットアップ

```bash
npm install
npm start
```

ブラウザで `http://localhost:8080` を開いてください。

## 使い方

1. 「話す・入力する言語」と「自分が読みたい言語」を選びます。
2. テキストを入力して「翻訳」を押すと翻訳されます。
3. 「会話に送信」を押すと、同じURLを開いている相手へ送信されます。
4. 音声で使う場合は「マイク開始」を押して話します。
5. 「音声認識後に会話へ自動送信」をオンにしておくと、話し終わった発話が自動翻訳されて会話へ送信されます。
6. 相手の発話は、自分が選んだ「自分が読みたい言語」に翻訳されて表示されます。
7. 発音練習をする場合は、入力テキストを入れて「お手本読み上げ」を押し、続けて「マイク開始」で同じ文を話します。
8. ネイティブ度、聞き取り精度、お手本一致がパーセントで表示されます。
9. 必要に応じて「読み上げ」や「コピー」を使えます。

## 注意

- マイク機能はブラウザの音声認識機能を使います。Chrome または Edge での利用がおすすめです。
- マイク権限を許可する必要があります。
- 翻訳はサーバー側の `/api/translate` から公開翻訳APIへリクエストします。公開APIの制限や通信状況により、翻訳に失敗する場合があります。
- 音声認識は `localhost` または HTTPS のページで動作します。
